# 📈 Marketing Campaign Analysis

## Project Overview

고객 특성 분석을 통해 소비 패턴을 파악하고, 이를 바탕으로 마케팅 전략을 제안한 프로젝트입니다.
총 소비액에 대한 회귀모델을 구축하고 Tableau를 활용해 핵심지표와 분석 결과를 시각화했습니다.

<br>

- Tableau

## Key Insight

- 소득은 소비액에 가장 큰 영향을 미치는 변수로 확인
- 캠페인 수락 경험이 많은 고객은 소비 규모가 큼
- 웹사이트 방문 횟수는 단순 평균과 회귀모형에서 다른 양상을 보여 고객 특성과 함께 해석할 필요가 있음을 확인
